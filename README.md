# Personal Portfolio + Blog Platform

## Project Idea
This project is a personal portfolio and blog platform where users can:

- View personal profile and skills
- Showcase projects and achievements
- Write and publish blog posts
- Contact through a contact form
- Responsive design for mobile and desktop

- CSS
- JavaScript
- Node.js
